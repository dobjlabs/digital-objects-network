# Digital Objects setup skill

This file is kept as a compatibility entry point for agents and older links.
`INSTALL.md` is the canonical setup guide.

Follow `INSTALL.md` to set up the Digital Objects driver on this machine.
Complete the agent path end to end:

1. Install the release binaries.
2. Start `dobjd`.
3. Install `craft-basics.pexe`.
4. Verify `dobj status`, `dobj actions`, and `dobj state-root`.
5. Register MCP for the current agent when supported.
6. Report exact command output for any failure.
