## Actions

| Action | Description |
| ------ | ----------- |
| `FindLog` | Discover a log object by proving a short VDF. |
| `CraftWood` | Refine one log into a wood object with PoW quality checks. |
| `CraftSticks` | Split one wood object into two stick objects. |
| `CraftWoodPick` | Combine wood and a stick to craft a wood pick. |
| `MineStoneWithWoodPick` | Mine stone using a wood pick (consumes durability). |
| `CraftStonePick` | Combine stone and a stick to craft a stone pick. |
| `MineStoneWithStonePick` | Mine stone using a stone pick (consumes durability). |
| `UseWoodPick` _(hidden)_ | Internal durability/work update for wood pick usage. |
| `UseStonePick` _(hidden)_ | Internal durability/work update for stone pick usage. |

## What it demonstrates

- **VDF-gated discovery** вЂ” `FindLog` proves an iterated-hash delay function,
  so new logs cost real wall-clock time.
- **PoW quality checks** вЂ” `CraftWood` grinds a key whose top limb must clear
  a difficulty target, proved with a 256-bit comparison.
- **Durability you can't fake** вЂ” using a pick proves the old durability was
  positive and that the new value is exactly one less.
- **Hidden subactions** вЂ” the `Use*Pick` helpers are composed into the mining
  actions rather than invoked directly.

## Install

Bundled with the [driver installer](/install). To add it to an existing
setup:

```bash
mkdir -p ~/.dobj/actions
curl -fsSL https://github.com/dobjlabs/zk-craft-releases/releases/latest/download/craft-basics.pexe \
  -o ~/.dobj/actions/craft-basics.pexe
~/.dobj/bin/dobj stop && ~/.dobj/bin/dobj start
```

Or from a checkout of the repo:

```bash
cargo run -p pexe --release -- build --install examples/craft-basics
~/.dobj/bin/dobj stop && ~/.dobj/bin/dobj start
```
